# AlterLynx  
В этом репозитории представлен проект браузера для альтернативной сети АльтерНет.
## Быстрый старт (Debian/Astra/Mint/Ubuntu)  
```
sudo apt install python3-commonmark zenity -y
git clone https://github.com/Michaelionin/alterlynx.git alterlynx
cd alterlynx
chmod +x launcher.sh
./launcher.sh
```
